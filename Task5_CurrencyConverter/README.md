# Currency Converter

## Description
This is a Java console-based Currency Converter application that allows users to convert amounts between different currencies using predefined exchange rates.

The application provides a simple menu-driven interface for selecting currencies and converting amounts.

## Features
- Convert USD to INR
- Convert INR to USD
- Convert USD to EUR
- Convert EUR to USD
- Convert INR to EUR
- Convert EUR to INR
- User-friendly console interface
- Input-based currency selection

## Technologies Used
- Java
- VS Code

## How to Run

Compile the program:

javac Main.java

Run the program:

java Main

## Sample Output
The program displays:
- Currency selection menu
- Amount input
- Converted currency value
