# Student Grade Calculator

## Description
This is a simple Java console-based application that calculates the total marks, average percentage, and grade of a student based on marks entered for multiple subjects.

## Features
- Input marks for multiple subjects
- Calculate total marks
- Calculate average percentage
- Display grade based on percentage
- Input validation for marks

## Technologies Used
- Java
- VS Code

## How to Run

Compile the program:

javac Main.java

Run the program:

java Main

## Sample Output
The program displays:
- Total Marks
- Average Percentage
- Grade
