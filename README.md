# CodSoft Java Internship Projects

This repository contains all the tasks completed as part of the CodSoft Java Internship using Java and VS Code.

The projects focus on core Java concepts such as object-oriented programming, file handling, user input validation, loops, conditional statements, and console-based application development.

---

## Completed Tasks

### Task 1 — Number Game
A simple number guessing game where the user tries to guess a randomly generated number with hints and score tracking.

### Task 2 — ATM Interface
A console-based ATM simulation system with features like:
- Balance checking
- Deposit money
- Withdraw money
- Input validation

### Task 3 — Student Management System
A Java application to manage student records with features such as:
- Add student
- Remove student
- Search student
- File handling using students.txt
- Data persistence

### Task 4 — Student Grade Calculator
A grade calculator application that:
- Accepts marks for multiple subjects
- Calculates total marks
- Calculates average percentage
- Displays grades

### Task 5 — Currency Converter
A console-based currency converter that converts:
- USD to INR
- INR to USD
- USD to EUR
- EUR to USD
- INR to EUR
- EUR to INR
