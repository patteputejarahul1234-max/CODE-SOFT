# Student Management System

## Description
This is a console-based Java application developed to manage student records efficiently.  
The project allows users to add, search, update, delete, and display student information through a menu-driven interface.

The application also stores student data permanently using file handling, so the records are saved even after closing the program.

---

## Features
- Add Student
- Display All Students
- Search Student by ID
- Update Student Details
- Delete Student
- Automatic Grade Calculation
- Input Validation
- File Handling using students.txt
- Menu-Driven Console Interface
